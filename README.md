# bnhp.io

Landing page for Bunny Hop — a small team building blockchain services and products.

## Stack

- Next.js (static export)
- Tailwind CSS v4
- JetBrains Mono
- GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Static files are exported to `out/`.

## Deploy

Push to `master` — GitHub Actions will build and deploy to GitHub Pages automatically.
